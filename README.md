### 上手指南


###### 开发前的配置要求

1. node > 14
2. npm 8

###### **安装步骤**

1. Clone the repo
2. npm install


### 主要文件目录说明
eg:

``` 
├── Blocks (Epub扩展积木)
├── components (Epub Blockly 组件)
├── store （Recoil 状态管理）
└── utils (辅助函数和功能)

```

### 开发的架构 

1. 基于Epub360 API
2. 制作积木

### 部署

1. npm run build
2. npm start
